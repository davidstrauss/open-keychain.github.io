Created by nuntius35, OpenKeychain developers Use the Space key to navigate through this tutorial.

In this HowTo you will learn

  • How to create your personal key,
  • how to find other people's keys,
  • how to setup K-9 Mail to work with OpenKeychain,
  • and finally how to send and receive encrypted emails.

Prerequisites

  • An Android device with an installed and configured K-9 Mail

Step 1: Installing OpenKeychain

  • Just install OpenKeychain from the Google Play Store.
  • If you don't have Google Play, you can also install OpenKeychain via F-Droid.

Step 2: Create your personal key

  • Only you can decrypt these emails with your personal key.
  • After creation, others can encrypt emails using this key and send them to you.

A friendly welcome screen

Just press Create My Key

Input your name

Input your email address

Input your password

Make sure it's a good one

Create the key...

No need to change the default setting.

... and wait ...

... and done!

Step 3: Export your personal key

  • Caution: You must not loose your personal key.
  • Therefore we are going to export it to a file which can be stored at a safe place.

Click on your key, select Export

Also select the checkbox to export the secret parts of the key. This file has to be stored at a safe place, e.g., on an USB stick.

Step 4: Import other people's keys

  • Before sending encrypted emails to an address, you need to get a key for this address.
  • OpenKeychain can download keys from the Internet.
  • To make sure that you get the right key you can scan other people's QR Codes or compare fingerprints (This will be part of a different HowTo).
  • For a simple test, we want to get the key for the address adele-en@gnupp.de.

Import key: You want to Search Cloud

Search in the cloud

Search for adele-en@gnupp.de, choose the key with Key ID e573 3469 92ab 3ff7 and press Import

If you are reading this tutorial on Android, click on this link: Get Adele's key

You have successfully imported a key

Step 5: Configure K-9 Mail

  • With a little help, it's easy.
  • Just follow us four levels deep into the menu.

Settings ...

... Account Settings ...

... Cryptography ...

... and now you choose OpenKeychain. And done.

Step 6: Compose your first encrypted mail to Adele

  • Adele is a friendly email robot.
  • You send an email with your public key.
  • She replies with an encrypted mail.
  • What are you waiting for?

Share your key from OpenKeychain

Open Extended information and share your key with K-9 Mail.

Compose email in K-9 Mail

Select sign and encrypt. Then send.

It's your first time: allow K-9 Mail access...

... and choose your own key.

This only have to be done once. Now everything is in place.

Signing accesses your personal key: password!

  • It's an exciting moment, isn't it?

Congratulations!
